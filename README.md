## Getting Started

### Prerequisites
- Python 3.11+

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/urban-pulpit.git
   cd urban-pulpit
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the app:
   ```bash
   streamlit run main.py
   ```
